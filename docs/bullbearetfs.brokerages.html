The Brokerate API Package represents the implementation of the Brokerage Component
It abstracts the communication between the Egghead platform and the individual brokerages.
It provides functionality such as: 
Connecting the platform to the Brokerage (Paper account, Live account)
Managing the connection to the Brokerage
Providing Account Information
Acquisition of pair symbols
Disposition of symbols
Order Execution/Management
Asset Protection
Retrieval of historical data
Account Information
